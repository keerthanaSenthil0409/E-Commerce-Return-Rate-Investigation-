# TASK 05: E-Commerce Return Rate Investigation

## Overview

This project analyzes e-commerce product return behavior using statistical and exploratory data analysis techniques. The objective is to identify the key factors influencing product returns and provide business recommendations to reduce return rates and improve customer satisfaction.

## Problem Statement

Product returns create significant challenges for e-commerce businesses, including increased logistics costs, inventory management issues, and revenue loss. This project investigates the factors contributing to returns and identifies patterns that can help businesses make data-driven decisions.

## Objectives

* Analyze e-commerce return data.
* Identify factors affecting product returns.
* Perform descriptive and statistical analysis.
* Detect outliers in customer and order behavior.
* Segment customers based on return patterns.
* Provide business insights and recommendations.

## Dataset Description

A synthetic e-commerce dataset containing **1000 customer orders** and **10 features** was used.

### Features

* Order_ID
* Customer_ID
* Order_Date
* Category
* Price
* Discount_Percent
* Delivery_Days
* Customer_Rating
* Region
* Returned

## Project Workflow

### Phase 1: Data Understanding

* Data loading and inspection
* Missing value analysis
* Duplicate removal
* Dataset exploration

### Phase 2: Descriptive Statistical Analysis

* Mean
* Median
* Standard deviation
* Quartiles
* Distribution analysis

### Phase 3: Return Pattern Investigation

* Category vs Return Rate
* Price vs Return Rate
* Discount vs Return Rate
* Delivery Time vs Return Rate
* Customer Rating vs Return Rate

### Phase 4: Outlier Analysis

* Box Plots
* IQR Method
* Z-Score Method

### Phase 5: Hypothesis Testing

* Null Hypothesis: Delivery delays do not affect returns.
* Alternative Hypothesis: Delivery delays significantly affect returns.
* Independent Sample T-Test.

### Phase 6: Customer Segmentation

* High-return customers
* Premium customers
* Discount-driven customers
* Delayed delivery customers
* Low-rating customers

### Phase 7: Business Insights

* Identified factors influencing returns.
* Determined risky product categories.
* Analyzed the impact of delivery delays.
* Recommended strategies to reduce return rates.

## Key Findings

* Delivery delays increase the likelihood of product returns.
* Higher discounts are associated with increased returns.
* Lower customer ratings are linked to higher return rates.
* Clothing has the highest return rate among product categories.
* Customer segmentation helps identify groups requiring business intervention.

## Business Recommendations

* Improve delivery efficiency.
* Enhance product quality and descriptions.
* Optimize discount strategies.
* Monitor high-return customers.
* Develop targeted customer retention strategies.
